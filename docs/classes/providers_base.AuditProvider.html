    Class AuditProviderAbstract

    Abstract base class for single-turn AI audit providers.

    Subclasses implement chat, parseResponse, ping, describe, and isConfigured. The concrete methods chatJSON and normalizeResponse are inherited and shared across all providers.

    Example

    const provider = createAuditProvider(config)
    const result = await provider.chatJSON(prompt, AuditResponseSchema)
