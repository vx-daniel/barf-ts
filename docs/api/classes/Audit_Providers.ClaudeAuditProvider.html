    Class ClaudeAuditProvider

    Audit provider backed by the Anthropic Claude Messages API. Extends AuditProvider — use createAuditProvider to instantiate.

    Uses the SDK directly for single-turn calls (not the subprocess approach in core/claude.ts, which is reserved for multi-turn streaming agent work).
