Add two new string utility functions to `src/index.ts`.

## Functions

**`capitalize(s: string): string`**
Uppercases the first character, lowercases the rest.
`capitalize('hELLO')` → `'Hello'`

**`titleCase(s: string): string`**
Capitalizes the first letter of each word (split on whitespace).
`titleCase('the quick brown fox')` → `'The Quick Brown Fox'`

## Acceptance Criteria

- [ ] `capitalize` uppercases first char and lowercases the rest
- [ ] `capitalize('')` returns `''`
- [ ] `titleCase` capitalizes first letter of every word
- [ ] `titleCase` handles multiple spaces between words
- [ ] Both functions are exported from `src/index.ts`
- [ ] Tests written in `tests/capitalize.test.ts`
