Add a `slugify` function that converts a string into a URL-safe slug.

## Function

**`slugify(s: string): string`**
- Lowercases the string
- Replaces spaces (and runs of whitespace) with `-`
- Removes any character that is not alphanumeric or `-`
- Collapses multiple consecutive `-` into one
- Strips leading and trailing `-`

Examples:
```
slugify('Hello World')       → 'hello-world'
slugify('  foo   bar  ')     → 'foo-bar'
slugify('It\'s a test!')     → 'its-a-test'
slugify('---hello---')       → 'hello'
```

## Acceptance Criteria

- [ ] Lowercases input
- [ ] Spaces become `-`
- [ ] Non-alphanumeric characters (except `-`) are removed
- [ ] Consecutive `-` are collapsed to one
- [ ] Leading/trailing `-` are stripped
- [ ] `slugify('')` returns `''`
