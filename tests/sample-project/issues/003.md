Add two counting utilities to `src/index.ts`.

## Functions

**`wordCount(s: string): number`**
Returns the number of words (split on whitespace, ignoring empty tokens).
`wordCount('  hello   world  ')` → `2`

**`charCount(s: string, opts?: { ignoreSpaces?: boolean }): number`**
Returns character count. When `ignoreSpaces: true`, whitespace is excluded.
`charCount('hello world')` → `11`
`charCount('hello world', { ignoreSpaces: true })` → `10`

## Acceptance Criteria

- [ ] `wordCount` counts words split by whitespace
- [ ] `wordCount` ignores leading/trailing/multiple spaces
- [ ] `wordCount('')` returns `0`
- [ ] `charCount` returns total character count by default
- [ ] `charCount` with `ignoreSpaces: true` excludes whitespace
- [ ] Both exported from `src/index.ts`
- [ ] Tests in `tests/counting.test.ts`
